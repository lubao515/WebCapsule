Webcapsule

Record and Replay Engine for Web Browsers

Download this project as a .zip file Download this project as a tar.gz file

Welcome to WebCapsule.

WebCapsule is a Record & Replay engine for Blink/V8. It allows for recording and replaying browsing traces, and primarily aims to enable the reconstruction and forensic analysis of web security incidents such as social engineering and phishing attacks.

You can find more information about WebCapsule by reading our ACM CCS 2015 paper, and our project wiki.
